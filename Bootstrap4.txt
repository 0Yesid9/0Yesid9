Clase 3 Ques es un framework de frontend
Tanto bootstrap como fundation son framework frontend (o framework css) y 
esto quiere decir que son una base para tener flexibilidad en el diseño de un 
proyecto web y una organizacion y estructura de nuestro Html, Css y javascript.
Con esto darle clases a los elementos de los estilos de css.
El componente principal es la Grilla: permite organizar el contenido de nuestro
sitio web y se adapta en diferentes dispositivos.
Los framework frontend traen estilos para las fuentes y componentes visuales 
pre-armados que se pueden utilizar para organizar la información.
*Ahorrar tiempo 
*componentes visuales
*Responsive design
*El codigo anda (no nos preocupamos por la compatibilidad con otros browser)
